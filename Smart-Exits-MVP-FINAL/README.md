# Smart Exits MVP

هذا المستودع يحتوي على النموذج الأولي (MVP) لمشروع **Smart Exits** لتوقع الازدحامات المرورية باستخدام المحاكاة + تعلم الآلة.

## مكونات المشروع
- **simulation/**  
  يحتوي كود المحاكاة، توليد البيانات، وتحليل الأنماط.
- **api/**  
  واجهة برمجية بسيطة (Flask) لتشغيل النموذج وعرض النتائج.
- **notebooks/**  
  دفاتر Jupyter للتحليل وتدريب النموذج.
- **assets/**  
  ملفات إضافية مثل الصور والمخططات.

## كيفية التشغيل
1. تثبيت المتطلبات:
   ```bash
   pip install -r requirements.txt
   ```

2. تشغيل واجهة Flask:
   ```bash
   python api/app.py
   ```

3. فتح دفاتر Jupyter:
   ```bash
   jupyter notebook
   ```

## الهدف
صناعة نموذج أولي يمكن تقييمه وتطويره إلى نموذج صناعي لاحقاً.
